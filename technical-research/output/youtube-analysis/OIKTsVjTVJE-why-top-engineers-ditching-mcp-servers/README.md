## Key Findings at a Glance

### The Problem
MCP servers consume 10,000+ tokens (5% of context) before agents even start working. Multiple servers can consume 20%+ of available context.

### The Solutions
1. CLI Tools (50% context savings)
2. File System Scripts (90% context savings)
3. Claude Skills (92%+ context savings)

### The Rule
80/15/5 - Use MCP 80% of time, CLI 15%, Scripts/Skills 5%

### The Principle
Build CLI-first, wrap MCP later. Prompt engineer before context engineer. Less garbage context = better performance.
