## Three-Step Pattern: Input-Workflow-Output

All effective agentic prompts follow this consistent structure:

```
INPUT: Define variables, parameters, static information
↓
WORKFLOW: Step-by-step executable instructions (most important)
↓
OUTPUT: Specify response format (JSON, YAML, markdown, etc.)
```

This pattern reduces cognitive load across all three stakeholder audiences.

## Control Flow in Natural Language

Complex programming concepts can be expressed naturally:
- **Conditionals**: "If X, then do Y, else do Z"
- **Loops**: "For each item in Z, perform Y"
- **Early Returns**: "Stop if insufficient information"

This democratizes agent programming to non-traditional developers.
