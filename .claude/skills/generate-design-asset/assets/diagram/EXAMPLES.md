# Diagram Style Examples

Examples showing diagram styles. Remember: **MAXIMUM 3 ELEMENTS**, abstract over literal.

---

## Two-Element Flow Example

**Context**: Explaining data transformation process

**Filled Prompt**:
```
Simple diagram showing two geometric elements connected by glowing line. Dark background.

Elements:
- Left: Glowing cyan cube with subtle internal glow
- Right: Glowing magenta sphere with surface shimmer
- Connection: Cyan particle stream flowing left to right

Labels: "DATA" on cube, "OUTPUT" on sphere (ALL CAPS).

Background: Deep black.

Technical specifications: 1920x1080, 16:9.
```

## Three-Element Process Example

**Context**: Three-stage agent workflow

**Filled Prompt**:
```
Linear diagram showing three hexagonal nodes connected in sequence. Dark background.

Elements left-to-right:
- Hexagon 1: Cyan
- Hexagon 2: Purple
- Hexagon 3: Green

Connections: White glowing lines connecting 1→2→3.

Labels: "INPUT", "PROCESS", "OUTPUT"

Background: Dark charcoal with subtle grid.

Technical specifications: 2560x1440, 16:9.
```
