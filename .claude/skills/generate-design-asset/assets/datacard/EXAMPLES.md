# DataCard Style Examples

Examples showing datacard styles with bold metrics and dramatic presentation.

---

## Metric Hero Example

**Context**: Performance improvement highlight

**Filled Prompt**:
```
Bold metric visualization with dramatic lighting. Number is hero.

Central element: "95%" rendered huge in bold sans-serif. Green (#4caf50) neon glow with strong bloom.

Background: Pure black with dramatic spotlight from upper-left.

Context: "FASTER" in smaller text below metric. Green glow.

Aesthetic: Bold, confident.

Technical specifications: 1200x1200, square aspect ratio.
```

**Use**: `performance-guide-datacard-95pct.png`

## Comparison Card Example

**Context**: Before/after optimization

**Filled Prompt**:
```
Split-screen comparison card. Dark background.

Left: "3 hours" in orange (#ff6600)
Right: "5 min" in green (#4caf50)

Vertical gradient split. Color zones reinforce contrast.

Labels: "BEFORE" (left), "AFTER" (right)

Aesthetic: Clear comparison, dramatic improvement.

Technical specifications: 1200x1600, portrait aspect ratio.
```
