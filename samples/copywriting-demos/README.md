# Copywriting Skills Demonstration Samples

This directory contains sample demonstrations of each copywriting skill in action, showing real edit suggestions generated by the skills.

## Sample Files

- **mad-men-demo.md** - Mad Men Copywriter suggestions on sales/landing page content
- **master-copy-demo.md** - Master Copywriter suggestions on thought leadership content
- **ameno-demo.md** - Ameno Voice suggestions on technical blog post content

## Purpose

These samples demonstrate:
1. How each skill classifies and analyzes content
2. The specific types of edits each skill suggests
3. The rationale format and quality expected
4. The before/after transformations for each skill's focus area
5. The suggest_edits output format

## Usage

Review these samples to understand:
- What Mad Men conversion-focused edits look like
- What Master Copywriter persuasion-focused edits look like
- What Ameno Voice simplification edits look like
- How to evaluate and apply suggestions selectively

## Note

These are example outputs. Actual skill output will vary based on:
- Input content quality and type
- Section types present
- Opportunities for improvement identified
- Skill-specific focus areas

Each sample includes 5-7 edit suggestions to show typical output volume and focus.
